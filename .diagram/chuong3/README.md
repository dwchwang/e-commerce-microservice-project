# Sơ đồ PlantUML - Chương 3

Thư mục này chứa sơ đồ tương ứng với placeholder hình ở Chương 3:

- `3_1_technology_overview.puml`: Hình 3.1, sơ đồ tổng quan các nhóm công nghệ trong hệ thống.

Render PNG:

```bash
plantuml .diagram/chuong3/*.puml
```

Render SVG để đưa vào LaTeX:

```bash
plantuml -tsvg .diagram/chuong3/*.puml
```
